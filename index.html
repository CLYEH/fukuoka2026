交通建議

以下內容已於 2026/04/21 依福岡市地下鐵、JR 九州、nimoca 與福岡市官方觀光網站核對。 黃金週期間售票櫃檯與巴士站人潮都可能較多，優先帶既有 IC 卡 會最省事。

最直接的結論

如果你已經有 Suica / PASMO / ICOCA 這類全國互通交通卡，直接帶去福岡用即可， 不需要特地再買一張新卡。福岡市地下鐵官方列出的可用卡已包含 HAYAKAKEN / nimoca / SUGOCA / Suica / PASMO / ICOCA 等。

沒有卡時買什麼

觀光使用上，HAYAKAKEN / nimoca / SUGOCA 都可以。重點不是卡名，而是買得到、可重複加值、轉車時能直接感應。

  • HAYAKAKEN：福岡市地下鐵可買，售價有 ¥1,000 / ¥2,000 / ¥3,000 / ¥4,000 / ¥5,000 / ¥10,000，皆含 ¥500 押金。
  • nimoca：一般卡售價 ¥2,000，其中 ¥500 為押金、¥1,500 可直接使用。
  • SUGOCA：無記名卡售價 ¥2,000，其中 ¥500 為押金、¥1,500 可直接使用。

只跑地鐵時

福岡市地下鐵一日券成人 ¥640 / 兒童 ¥320，限當日使用。 地鐵普通票目前依區間為 ¥210–¥380，所以如果當天會頻繁搭地下鐵， 大致搭到 3 趟左右就可以開始評估一日券。

巴士 + 地鐵 + 太宰府很多段時

FUKUOKA TOURIST CITY PASS 適合把巴士、地鐵、西鐵、渡輪混搭在同一天的人。 福岡市版成人 ¥2,500 / 兒童 ¥1,250；福岡市 + 太宰府版成人 ¥2,800 / 兒童 ¥1,400。 紙本票需出示護照，且限海外短期停留旅客購買；數位版可用 my route， 但官方目前註明僅開放韓國、台灣、泰國旅客下載使用。

注意：官方註明此票不適用 Dazaifu Liner Bus「Tabito」。

這次 5 天 4 夜怎麼買最合理

Day 1：機場、博多、LaLaport、天神之間交通型態混合，建議直接刷 IC 卡，最彈性。

Day 2：主要是一日團，只需要處理天神 ⇄ 博多移動，IC 卡就夠。

Day 3：目前行程以西鐵 + 地下鐵為主，通常先用 IC 卡即可；若之後又加很多市區巴士景點，再回頭評估觀光 pass。

Day 4–5：海之中道與回機場以 JR / 地鐵為主，仍以 IC 卡最方便。
